# OpsFlow Device Screen Mockup Template Guide\n\n## Figma Template Structure\n\n### Base Components\n\n1. **Tablet Frame (1024x768)**\n   - **Outer Frame**: 1044x788 (10px bezel on all sides)\n   - **Screen Area**: 1024x768 (actual UI content area)\n   - **Colors**: \n     - Bezel: #2c3e50 (dark gray)\n     - Screen: #000000 (black, will be replaced with UI)\n   - **Corner Radius**: 8px outer, 4px inner screen\n\n2. **Kitchen Wall Mount** \n   - **Mount Bracket**: Simple wall mounting bracket behind tablet\n   - **Shadow**: Subtle drop shadow (0px 4px 12px rgba(0,0,0,0.15))\n   - **Wall Texture**: Subtle stainless steel texture background\n\n3. **Screen Overlay Effects**\n   - **Glare Gradient**: Linear gradient from top-left\n     - Start: rgba(255,255,255,0.1) at 0%\n     - End: rgba(255,255,255,0.02) at 100%\n   - **Reflection**: Subtle white highlight on top edge\n   - **Fingerprint Smudges**: Very subtle texture overlay (optional)\n\n### UI Content Templates\n\n#### Temperature Dashboard\n```\nLayout: 3-column grid\nLeft: Temperature readings with color-coded status\nCenter: Historical chart/graph\nRight: Alert notifications and compliance status\n\nColors:\n- Background: #f8fafc\n- Hot (>60°C): #dc2626 \n- Cold (<5°C): #2563eb\n- Normal: #16a34a\n- Warning: #d97706\n```\n\n#### Task Checklist\n```\nLayout: Single column list\nHeader: \"Opening Checklist - 6:00 AM\"\nItems: Checkbox + task description + time stamp\nProgress bar at top showing completion %\n\nColors:\n- Completed: #16a34a\n- Pending: #64748b\n- Overdue: #dc2626\n- Background: #ffffff\n```\n\n#### Staff Schedule\n```\nLayout: Grid/calendar view\nRows: Staff members\nColumns: Time slots\nShift blocks with color coding by role\n\nColors:\n- Kitchen Staff: #3b82f6\n- Servers: #8b5cf6  \n- Managers: #059669\n- Background: #f1f5f9\n```\n\n## Photoshop Composite Workflow\n\n### Layer Structure (Top to Bottom)\n1. **Screen Glare** (Blend: Soft Light, 15% opacity)\n2. **UI Content** (Blend: Normal, 100% opacity)\n3. **Screen Bezel** (Inner shadow for depth)\n4. **Tablet Frame** (Outer bezel with texture)\n5. **Drop Shadow** (Multiply, 20% opacity)\n6. **Background** (Kitchen environment)\n\n### Step-by-Step Process\n\n1. **Import AI Image**\n   - Open AI-generated image with neutral tablet screen\n   - Duplicate background layer for safety\n\n2. **Create UI Layer**\n   - Import OpsFlow UI design as PNG (transparent background)\n   - Use **Edit > Transform > Perspective** to match tablet angle\n   - Use **Edit > Transform > Distort** for fine adjustments\n\n3. **Apply Screen Effects**\n   - Create new layer above UI\n   - Add linear gradient (white to transparent)\n   - Set blend mode to \"Screen\" or \"Linear Dodge\"\n   - Reduce opacity to 10-15%\n\n4. **Add Realistic Details**\n   - Subtle inner shadow on screen bezel\n   - Very light reflection on top edge\n   - Optional: Add subtle texture for fingerprints\n\n5. **Color Grading**\n   - Apply brand LUT (cool temperature, +200K)\n   - Enhance blues in UI content\n   - Maintain commercial kitchen lighting\n\n### Export Settings\n- **Master**: PSD file (layered for easy updates)\n- **Web**: WebP, 85% quality, sRGB color space\n- **Backup**: PNG-24 for maximum compatibility\n\n## UI Design Guidelines for Device Screens\n\n### Typography\n- **Primary Font**: Inter (web-safe alternative: system-ui)\n- **Sizes**: \n  - Headers: 24-32px\n  - Body: 16-18px (tablet optimized)\n  - Labels: 14px\n  - Data: 20-28px (highly legible)\n\n### Touch Targets\n- **Minimum Size**: 44x44px (Apple HIG standard)\n- **Preferred Size**: 48x48px for kitchen environment\n- **Spacing**: 8px minimum between interactive elements\n\n### Color Accessibility\n- **Contrast Ratio**: Minimum 4.5:1 for normal text\n- **Large Text**: Minimum 3:1 contrast\n- **Interactive Elements**: 3:1 contrast with adjacent colors\n\n### Content Density\n- **Maximum Items**: 6-8 items per screen (kitchen readability)\n- **White Space**: 24px margins, 16px between sections\n- **Scan Pattern**: Left-to-right, top-to-bottom flow\n\n## Brand-Specific UI Elements\n\n### OpsFlow Button
